How it works

This server automatically injects a small WebSocket script into your HTML pages. When any file changes, the browser instantly reloads to show your updates.

Try it out!

Open this file in your editor and make some changes. Save the file and watch the browser automatically reload!

Usage

# Install dependencies
pip install -r requirements.txt

# Run the server
python src/server.py

# Or with options
python src/server.py --port 8080 --directory ./my-site
